1 [5점]. 클라이언트는 대화상자(Dialog)를 이용하고, 서버는 콘솔프로그램 또는 윈도우 프로그램 중 골라서 작성하시오. 
[수업시간에 설명한 윈도우 기반 채팅 프로그램을 수정하여 작성하여도되며, 소켓의 변화 감지에 Select, WSAAsyncSelect, WSAEventSelect, 11장의 소켓 입출력 모델을 이용하여 됨]

- 대화방을 두 개를 구성하시오.

- 각 방내에서 다자간 채팅이 되도록 구현하시오.

- IP주소와 Port번호, 사용자명, 방 번호는 클라이언트에서 입력 값을 받도록 하며, 잘 못 입력할 시 오류 처리를 하도록 하시오.

- 각 방별로 사용자명을 다르게 설정할 수 있다.

- 채팅창에는 사용자명과 대화내용이 출력되면 된다.

2. [1점] 접속해 있는 사용자를 원할 때 Display할 수 있도록 하시오.

2개의 대화방에 있는 사용자를 전부 출력하여 볼수 있도록 하시오.

3. [2점] 대화방에 들어갈 때 이미 접속해 있는 사용자명을 입력하는 경우 경고메시지를 받고 사용자 명을 변경하여 접속하게 하도록 구현하시오.

4. [2점] 어떻게 프로그램을 설계하였는 지와 실행 예에 대하여 리포트를 작성하시오.
